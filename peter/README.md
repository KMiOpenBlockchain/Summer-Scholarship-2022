# Token Minter Management Demo

## Overview
This folder contains the **Token Minter Management** demo created as part of the **KMi Summer Scholarship Blockchain Project (2022)**.

The demo showcases how to:
- Connect to MetaMask
- Manage TikTok Token Minters (add, remove, verify)
- Interact with an Ethereum smart contract using client-side JavaScript and the Ethers.js library

## Notes
- This is a **client-side only** proof-of-concept, intended for educational purposes during the scholarship.
- No server-side code or backend services are included.
